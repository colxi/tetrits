# tetrits

Basic browser-based implementation of the Tetris game, using Typescript.

![alt text](https://github.com/colxi/tetrits/blob/main/assets/tetrits.jpg?raw=true)

## Features :

- Compact: 16Kb
- Keyboard control
- Block movement
- Block rotation
- Block projection
- Basic scoring system
- Basic leveling system
- Pause/resume
